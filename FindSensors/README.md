# Скетч поиска адресов сенсоров DS18B20

Здесь вы найдете простой скетч, который ищет адреса сенсоров DS18B20. Скетч может использоваться с любой платой Arduino/ESP32.  

Настойки Arduino IDE зависят от сипользуемой платформы. По умолчанию, используемые пины такие же, как и в основном скетче термометра в теплицу. Если вы используете другую плату, то переопределите пины. Так же закоментируйте строку #define _USE_POWER_PIN_.  

**Используемые библиотеки Arduino**

- esp32-ds18b20 2.0.3
